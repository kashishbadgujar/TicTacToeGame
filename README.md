# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using React JS. Players take turns placing X and O on the board to win the game.

## 🚀 Features

- 🎮 Two-player gameplay
- ❌⭕ Dynamic player turns
- 🏆 Winner detection
- 🤝 Draw game detection
- 🔄 Restart game functionality

## 🎮 Game Rules

- The game is played between two players: X and O.
- Players take turns selecting empty cells on the board.
- The first player to align 3 symbols horizontally, vertically, or diagonally wins.
- If all cells are filled and no player wins, the game ends in a draw.
- After the game ends, players can restart and play again.

## 🛠️ Tech Stack

- React JS
- JavaScript
- CSS3
- Vite

## ▶️ How to Run

1. Clone the repository
 
```bash
git clone https://github.com/kashishbadgujar/TicTacToeGame.git
```

2. Install dependencies

```bash
npm install
```

3. Start development server

```bash
npm run dev
```

4. Open in browser

```bash
http://localhost:5173
```
